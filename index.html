PySheetsQL Documentation

Overview

pysheetsql is a Python package for managing Google Sheets data using the gspread library. It provides an easy-to-use interface for creating, updating, retrieving, deleting, and exporting Google Sheets data.

Installation

To install pysheetsql, use pip:

pip install pysheetsql

Getting Started

Google API Credentials

To use pysheetsql, you need to obtain Google Sheets API credentials. Follow these steps to get your credentials.json file:

  1. Create a Google Cloud Project:
    • Visit the Google Cloud Console.
    • Click on "Select a project" and then "New Project".
    • Enter a project name and click "Create".
  2. Enable APIs:
    • Navigate to the API Library.
    • Search for and enable the Google Sheets API.
    • Also, enable the Google Docs API.
  3. Create Credentials:
    • Go to the Credentials page.
    • Click on "Create Credentials" and select "Service account".
    • Follow the prompts to set up the service account.
    • After creating the service account, click "Create Key" and choose JSON format. This will download your credentials.json file.

Setup

Create a SheetClient object to interact with Google Sheets:

from pysheetsql.start import SheetClient

client = SheetClient(
    scopes=["https://www.googleapis.com/auth/spreadsheets"],
    credentials_file_path="path/to/credentials.json"
)

Replace "path/to/credentials.json" with the actual path to your downloaded credentials.json file.

Functions

CreateTable

Creates a table in Google Sheets.

from pysheetsql.table import CreateTable

CreateTable(client, "StudentRecords", ["Name", "Age", "Grade"])

DeleteTable

Deletes a table from Google Sheets.

from pysheetsql.table import DeleteTable

DeleteTable(client, "StudentRecords")

ListTables

Lists all tables available in your Google Sheets account.

from pysheetsql.table import ListTables

ListTables(client)

ExportTable

Exports a table to a CSV file.

from pysheetsql.table import ExportTable

ExportTable(client, "StudentRecords")

AddData

Adds data to a table in Google Sheets.

from pysheetsql.manage_table import AddData

AddData(client, "StudentRecords", ["John Doe", 20, "A"])

GetDataName

Retrieves a row of data from a table based on the main field name.

from pysheetsql.manage_table import GetDataName

data = GetDataName(client, "StudentRecords", "John Doe")
print(data)

GetDataID

Retrieves a row of data from a table based on the primary ID.

from pysheetsql.manage_table import GetDataID

data = GetDataID(client, "StudentRecords", 1)
print(data)

UpdateDataName

Updates a value in a row based on the main field name.

from pysheetsql.manage_table import UpdateDataName

UpdateDataName(client, "StudentRecords", "John Doe", "Age", "21")

UpdateDataID

Updates a value in a row based on the primary ID.

from pysheetsql.manage_table import UpdateDataID

UpdateDataID(client, "StudentRecords", 1, "Age", "22")

DeleteData

Deletes a row based on the main field name.

from pysheetsql.manage_table import DeleteData

DeleteData(client, "StudentRecords", "John Doe")

GetAllData

Retrieves all rows of data from a table as a list of lists.

from pysheetsql.manage_table import GetAllData

data = GetAllData(client, "StudentRecords")
print(data)
