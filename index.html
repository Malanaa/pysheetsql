Overview

pysheetsql is a simple Python library I made to automate logging data to Google Sheets. I have designed the library to perform basic CRUD operations and use it as a small-scale database for static side projects.

Installation

To install pysheetsql, use pip:

pip install pysheetsql

Getting Started

Google API Credentials

To use pysheetsql, you need to obtain Google Sheets API credentials. Follow these steps to get your credentials.json file:

  1. Create a Google Cloud Project:
    • Visit the Google Cloud Console.
    • Click on "Select a project" and then "New Project".
    • Enter a project name and click "Create".
  2. Enable APIs:
    • Navigate to the API Library.
    • Search for and enable the Google Sheets API.
    • Also, enable the Google Docs API.
  3. Create Credentials:
    • Go to the Credentials page.
    • Click on "Create Credentials" and select "Service account".
    • Follow the prompts to set up the service account.
    • After creating the service account, click "Create Key" and choose JSON format. This will download your credentials.json file.

Setup

Create a SheetClient object to interact with Google Sheets:

from pysheetsql.start import SheetClient

client = SheetClient(
    scopes=["https://www.googleapis.com/auth/spreadsheets"],
    credentials_file_path="path/to/credentials.json"
)

Replace "path/to/credentials.json" with the actual path to your downloaded credentials.json file.

Functions

CreateTable

Creates a table in Google Sheets.

def CreateTable(client, table_name, fields, primary_key=True):
    """
    Creates a table in Google Sheets.

    Args:
        client (SheetClient): The SheetClient client instance.
        table_name (str): The name of the table.
        fields (list of str): A list of field names (the first field is considered the class name).
        primary_key (bool, optional): Whether to include a primary key field. Default is True.

    Returns:
        None: The function does not return a value.

    Raises:
        TypeError: If `table_name` is not a string or `fields` is not a list of strings.

    Example:
        CreateTable(client, "StudentRecords", ["Name", "Age", "Grade"])
    """

DeleteTable

Deletes a table from Google Sheets.

def DeleteTable(client, table_name):
    """
    Deletes a table from Google Sheets.

    Args:
        client (SheetClient): The SheetClient client instance.
        table_name (str): The name of the table you want to delete.

    Returns:
        None: The function does not return a value.

    Raises:
        TypeError: If `table_name` is not a string.
        gspread.exceptions.SpreadsheetNotFound: If the table does not exist.

    Example:
        DeleteTable(client, "StudentRecords")
    """

ListTables

Lists all tables available in your Google Sheets account.

def ListTables(client):
    """
    Lists all the tables available in your Google Sheets account.

    Args:
        client (SheetClient): The SheetClient client instance.

    Returns:
        list of str: A list of table names.

    Example:
        tables = ListTables(client)
        print(tables)
    """

ExportTable

Exports the data from a specified table to a CSV file.

def ExportTable(client, table_name, file_path):
    """
    Exports the data from a specified table to a CSV file.

    Args:
        client (SheetClient): The SheetClient client instance.
        table_name (str): The name of the table to export.
        file_path (str): The file path to save the CSV file.

    Returns:
        None: The function does not return a value.

    Raises:
        TypeError: If `table_name` or `file_path` is not a string.

    Example:
        ExportTable(client, "StudentRecords", "exported_data.csv")
    """

AddData

Adds a new row of data to a table.

def AddData(client, table_name, data):
    """
    Adds a new row of data to a table.

    Args:
        client (SheetClient): The SheetClient client instance.
        table_name (str): The name of the table to add data to.
        data (list of str): A list of data values corresponding to the table's fields.

    Returns:
        None: The function does not return a value.

    Raises:
        TypeError: If `table_name` is not a string or `data` is not a list of strings.

    Example:
        AddData(client, "StudentRecords", ["John Doe", "15", "A"])
    """

Sample Project

Here is a simple example project that demonstrates the usage of pysheetsql:

from pysheetsql.start import SheetClient
from pysheetsql.table import CreateTable, DeleteTable, ListTables, ExportTable
from pysheetsql.manage_table import AddData, GetAllData, GetDataID, GetDataName, getID, DeleteData, UpdateDataID, UpdateDataName

# Initialize the client
client = SheetClient(
    scopes=["https://www.googleapis.com/auth/spreadsheets"],
    credentials_file_path="path/to/credentials.json"
)

# Create a table
CreateTable(client, "SampleTable", ["ID", "Name", "Value"])

# Add data to the table
AddData(client, "SampleTable", ["1", "Sample Item", "123"])

# List tables
tables = ListTables(client)
print(tables)

# Export table data
ExportTable(client, "SampleTable", "sample_data.csv")

# Delete the table
DeleteTable(client, "SampleTable")
