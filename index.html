pysheetsql Documentation

Importing the Package

To use the functions provided in the pysheetsql package, first import the necessary modules. Here is how you can import and use them:


from pysheetsql.start import SheetClient
from pysheetsql import table, manage_table

# Initialize the client
client = SheetClient(["https://www.googleapis.com/auth/spreadsheets"], "path/to/credentials.json")

CreateTable

Creates a new table in Google Sheets. If the table already exists, it will notify you.


# Create a table
table.CreateTable(client, "StudentRecords", ["Name", "Age", "Grade"])

DeleteTable

Deletes an existing table. If the table does not exist, it will notify you.


# Delete a table
table.DeleteTable(client, "StudentRecords")

ListTables

Lists all the tables (spreadsheets) available in your Google account.


# List all tables
table.ListTables(client)

ExportTable

Exports a specified table as a CSV file to your local directory.


# Export a table to CSV
table.ExportTable(client, "StudentRecords")

AddData

Adds data to an existing table. If the table has a primary key, ensure to provide data excluding the primary key value.


# Add data to a table
manage_table.AddData(client, "StudentRecords", ["John Doe", 20, "A"])

GetAllData

Retrieves all data from the specified table.


# Get all data from a table
data = manage_table.GetAllData(client, "StudentRecords")
print(data)

GetDataID

Retrieves data by ID from the specified table.


# Get data by ID
data = manage_table.GetDataID(client, "StudentRecords", 1)
print(data)

GetDataName

Retrieves data by name from the specified table.


# Get data by name
data = manage_table.GetDataName(client, "StudentRecords", "John Doe")
print(data)

UpdateDataID

Updates data in the table by ID.


# Update data by ID
manage_table.UpdateDataID(client, "StudentRecords", 1, ["Jane Doe", 21, "B"])

UpdateDataName

Updates data in the table by name.


# Update data by name
manage_table.UpdateDataName(client, "StudentRecords", "John Doe", ["John Doe", 21, "A+"])

DeleteData

Deletes data from the table by ID or name.


# Delete data by ID
manage_table.DeleteData(client, "StudentRecords", 1)

# Delete data by name
manage_table.DeleteData(client, "StudentRecords", "John Doe")
